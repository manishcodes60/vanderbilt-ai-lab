On this website we explain the coding scheme that we use and how to go through the coding process. We do not assume that you have any specialized technical skills, so we provide step-by-step instructions. We have strived to ensure that you will have minimal computing costs (less than $100).

Technical Skills

We have designed the interface with our AI model in a way that does not require you to have technical expertise related to programing or LLMs. We provide a window where you post your transcript, and give you results in a spreadsheet format that includes each line of text, with the assigned code. The only technical skills you need are to put the transcript into the right format before pasting into the analysis window.

Registration for AI models

You do not need to have registered for any AI models, or to have an account on Github. That part is done by us.

Cost

Computing costs should be less than $100 for coding fifty 30-minute transcripts. (Or: At the moment, we have funds to cover the compute costs. If there is later a need for users to cover compute costs, it should be less than $100 for coding fifty 30-minute transcripts.)
